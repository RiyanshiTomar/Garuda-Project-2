Walk through the core features list so students know the full scope:

• User Registration & Login — with hashed passwords and JWT tokens
• Secure Authentication — every protected route verifies your token
• Wallet System — each user has a balance they can top up
• Send Money — transfer funds between users atomically (all-or-nothing)
• Transaction History — see all past credits and debits
• Email Notifications — nodemailer sends emails on key events
• Input Validation & Error Handling — because bad data breaks production

what is mvc now 

MVC (Model View Controller) is a software architecture pattern that separates an application into three main parts:

Model → handles data and database operations
View → handles the user interface (UI)
Controller → handles requests and controls the flow between View and Model
